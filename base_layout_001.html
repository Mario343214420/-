CSS知识点

position定位 定位与居中 Flex伸缩盒模型

`static`
是默认值
`relative`
相对定位 相对于自身原有位置进行偏移，仍处于标准文档流中
`absolute`
绝对定位 相对于最近的已定位的祖先元素, 有已定位(指`position`不是`static`的元素)祖先元素, 以最近的祖先元素为参考标准。如果无已定位祖先元素, 以`body`元素为偏移参照基准, 完全脱离了标准文档流。
`fixed`
固定定位的元素会相对于视窗来定位,这意味着即便页面滚动，它还是会停留在相同的位置。一个固定定位元素不会保留它原本在页面应有的空 问题：使用脱离文档流定位会出现父标签高度塌陷,需为父级标签元素添加高宽
方案1
position+transform居中（或负margin）
.son { position: absolute; left: 50%; transform: translate(-50%, 0); }
.son { position: absolute; width: 宽度; left: 50%; margin-left: -0.5*宽度 }
.son { position: absolute; width: 宽度; left: 0; right: 0; margin: 0 auto; }
.son { position: absolute; top: 50%; transform: translate( 0, -50%);}
方案2
text-align+line-height
.parent { height: 高度; }
.son { line-height: 高度; }
方案3
width+margin
.center-block { margin: 0 auto; }
方案4
flex--早期display：box
.parent { display: flex; align-items: center; }
方案5
table居中
.parent { display: table; }
.son { display: table-cell; vertical-align: middle; }
这一css布局能够实现良好的适应性弹性布局，并且css配置项完善。 Flex详细说明
  • flex-direction（常用）：设置横向排列或纵向
  • flex-wrap（常用）：设置换行
  • justify-content（常用）：设置间隔形式及对其方式
  • align-items（常用）：设置排列顺序
